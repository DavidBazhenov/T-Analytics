Так хорошо, теперь задача для тебя написать продвинутый генератор данных для датафрейма, 
который будет включать зарплату и аванс. Так же на основе финансовых возможностей человека, 
то есть его зарплаты и аванса, будет генерировать список его трат. 
Структура для нашего датафрейма должна выглядеть вот таким образом : 
data = {   
    'Date' : ['2024-01-01', '2024-01-02', ...],
    'Amount' : [100000, -505.3, -100, 400 ...],
    'Category' : ['Salary', 'Transport', 'Transact to', 'Transact from']
}
Длины массивов по ключам должны 'Date', 'Amount', 'Category' должны быть одинаковыми.
В один и тот же день допускается несколько трат, каждая трата должна иметь отрицательное значение
в столбце Amount. Категория 'Transact to' подразумевает перевод кому-то другому на счет,
то есть уменьшает баланс счета. Категория 'Transact from' подразумевает перевод от кого-то другого на счет,
то есть увеличивает баланс карты. Давай по минимум будем использовать категории 'Transact to', 'Transact from'.
Каждый месяц должны быть поступления по зарплате 'Salary' и по авансу 'Advance' минимум 1 раз, максимум 3 раза.
Категории трат можешь придумать достаточно много. Название категории должно начинаться с большой буквы.
По требованием это все. Генератор должен быть написан на python.å
